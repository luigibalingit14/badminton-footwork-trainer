How to Use

Practice Mode

Train your footwork by moving to highlighted zones. Adjust the pause time between shots to match your skill level.

  • Zones will light up one at a time
  • Move to the highlighted zone as quickly as possible
  • Return to center after each shot

Rally Mode

Simulate real game scenarios with burst sequences of shots followed by rest periods.

  • Set the number of shots per rally (±30% variation)
  • Zones will highlight rapidly in sequence
  • Rest during the pause between rallies

Tips

  • Always return to center position between shots
  • Focus on proper footwork technique, not just speed
  • Start with longer pauses and gradually decrease
  • Use Rally mode to build endurance
