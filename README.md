# Badminton Footwork Trainer

Interactive badminton footwork training application with Practice and Rally modes.

## Features

- **Practice Mode**: Train footwork with adjustable pause times between shots
- **Rally Mode**: Simulate game scenarios with:
  - Adjustable shots per rally (±30% variation)
  - Speed control for shot frequency
  - Visual shot counter
  - Voice countdown between rallies
- **Zone Management**: Click zones to enable/disable during training
- **Drag & Drop**: Reposition zones on the court
- **Voice Announcements**: Hear zone numbers called out
- **Responsive Design**: Works on desktop and mobile devices

## How to Use

1. Choose Practice or Rally mode
2. Adjust settings using + and - buttons
3. Click zone numbers to enable/disable them (✓ or ✗)
4. Press START to begin training
5. Move to highlighted zones as they light up
6. Press STOP or ESC to end session

## Technologies Used
